# MythicSMP
a plugin for survival minecraft servers by EplayYT 
Minecraft survival plugin

## Komutlar(commands)
- /myth <hikaye> > Efsane oluşturur TR
- /mythinfo > Efsaneyi gösterir TR

## Kurulum
1. JAR ı plugins klasörüne at TR
2. Sunucuyu başlat TR
3. by EplayYT for java 17+ minecraft servers(TR EplayYT tarafından, java 17 minecraft sunucuları için yapıldı
